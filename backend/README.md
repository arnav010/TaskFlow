# TaskFlow

TaskFlow is a simple beginner-level full-stack learning project. It uses HTML, CSS, and vanilla JavaScript on the frontend, and Node.js, Express.js, MongoDB, and Mongoose on the backend.

This version is intentionally small. It does not include React, authentication, AI features, drag and drop, charts, filters, tests, or deployment setup.

## Project Overview

TaskFlow allows you to:

- Create and delete boards
- Add and delete tasks
- View tasks in three columns: To Do, In Progress, and Done
- Move tasks between columns using simple buttons

The project is meant for students who are learning basic CRUD operations and full-stack folder organization.
