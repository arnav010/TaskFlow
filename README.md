# TaskFlow

## Project Description

TaskFlow is a beginner-friendly full-stack task management web application developed using **HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB**. The application enables users to organize their work by creating project boards and managing tasks within each board through a simple and intuitive interface.

The project demonstrates the implementation of RESTful APIs, CRUD operations, database integration using MongoDB, and communication between the frontend and backend. It is designed to help beginners understand the fundamentals of full-stack web development while building a practical real-world application.
